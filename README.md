# 🚀 CcodeLearner.com – Programming Learning Platform

## 📌 Overview
CcodeLearner.com is an interactive web-based programming learning platform designed to provide accessible, structured, and engaging educational content for learners at all levels. The platform includes tutorials, coding examples, quizzes, and blogs to enhance programming skills.

## ⚙️ Installation & Setup

### Prerequisites
- PHP >= 7.4
- MySQL Database
- Apache/Nginx Server
- WordPress Installed

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ccodelearner.git
````

2. Move project to server directory:

   ```bash
   /var/www/html/
   ```

3. Configure database in `wp-config.php`:

   ```php
   define('DB_NAME', 'your_db');
   define('DB_USER', 'your_user');
   define('DB_PASSWORD', 'your_password');
   define('DB_HOST', 'localhost');
   ```

4. Run the project:

   ```
   http://localhost/ccodelearner
   ```
